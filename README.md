# table maker

## install

- > npm install tablemaker

## usage

```typescript

	import { TableMaker } from 'tablemaker';

 const data=[
        { name: 'hero', age: '19' },
        { name: 'vilain', age: '50' },
        { name: 'vallager', age: '20' },
    ]
	
	
    <TableMaker attributes={data} />
	
	


```
